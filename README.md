Описание работы:

В рамках задания была разработана демонстрационная веб-страница с использованием возможностей HTML5 API.

Реализованы следующие функции:

History API
Реализована навигация по истории браузера с использованием history.pushState.
Добавлены кнопки перехода между страницами («Следующая страница», «Предыдущая страница»), а также кнопки управления историей браузера («Назад», «Вперед»).
Обработчик onpopstate обеспечивает корректное обновление состояния страницы при навигации.

Canvas API и анимация
Создана интерактивная анимация на элементе <canvas>.
Объекты (круги) перемещаются по холсту и реагируют на положение курсора мыши.
Для плавности анимации используется requestAnimationFrame.

Web Workers
Реализован веб-воркер для выполнения ресурсоёмких вычислений в отдельном потоке.
Это позволяет избежать блокировки пользовательского интерфейса.
Обмен данными осуществляется через postMessage.

Дополнительные требования
Создан пользовательский интерфейс для взаимодействия со всеми функциями приложения.
Реализована обработка ошибок (например, при отсутствии введённых данных).
Обеспечена отзывчивость интерфейса за счёт использования Web Worker и оптимизации анимации.
